# The Creator

The Creator is a game engine for pervasive and physically distributed games.

More to follow.

## Requirements

The Creator is tested on MRI Ruby 1.9.2/1.9.3 running Rails 4.0 on a mysql (5.6.13). 

It should run fine on any similar system.
